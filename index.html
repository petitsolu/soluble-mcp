URL du serveur (Transport SSE)
https://mcp.csoluble.media/api/mcp

🚀 Guide d'installation

1. Cursor & Windsurf (Recommandé)

Ouvrez les réglages MCP et ajoutez un serveur de type SSE avec ce bloc JSON :

{
  "mcpServers": {
    "solubles": {
      "url": "https://mcp.csoluble.media/api/mcp"
    }
  }
}

2. Claude Desktop (Mac / Windows)

Ajoutez ce bloc à votre fichier claude_desktop_config.json :

{
  "mcpServers": {
    "solubles": {
      "url": "https://mcp.csoluble.media/api/mcp"
    }
  }
}

3. Claude Code (Terminal)

Une seule commande à lancer dans votre terminal :

claude mcp add solubles --transport sse https://mcp.csoluble.media/api/mcp
