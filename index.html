🚀 Guide d'installation

1. Cursor / Windsurf

Allez dans Settings > Tools > MCP. Ajoutez un nouveau serveur avec ces paramètres :
• Type : SSE
• URL : https://mcp.csoluble.media/api/mcp

2. Claude Desktop

Ajoutez ce bloc à votre fichier claude_desktop_config.json :

"mcpServers": {
  "solubles": {
    "command": "npx",
    "args": [
      "-y", 
      "@modelcontextprotocol/inspector", 
      "--transport", 
      "sse", 
      "https://mcp.csoluble.media/api/mcp"
    ]
  }
}
