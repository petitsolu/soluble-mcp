Connectez Soluble(s) à votre Intelligence Artificielle

Interrogez directement notre base de solutions concrètes depuis Claude, Cursor ou Windsurf grâce au protocole MCP.

URL du serveur officiel
https://mcp.csoluble.media/api/mcp

🚀 Installation Rapide

Dans Cursor ou Windsurf

1. Allez dans Settings > Tools > MCP.
2. Ajoutez un nouveau serveur (Type: SSE).
3. Collez l'URL ci-dessus.

Dans Claude Desktop

Modifiez votre fichier claude_desktop_config.json :

"mcpServers": {
  "solubles": {
    "command": "npx",
    "args": ["-y", "@modelcontextprotocol/inspector", "--transport", "sse", "https://mcp.csoluble.media/api/mcp"]
  }
}
