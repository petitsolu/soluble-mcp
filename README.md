# Soluble(s) MCP

**Un connecteur IA simple et utile pour le podcast Soluble(s)**  
**A simple and useful AI connector for the Soluble(s) podcast**

Créé par **Simon Icard** – journaliste indépendant à Marseille.  
Created by **Simon Icard** – independent journalist in Marseille.

### Comment l’utiliser (30 secondes) / How to use (30 seconds)

**Français**  
1. Ouvre **Claude** (claude.ai ou Desktop)  
2. Paramètres → Connectors → Add custom connector  
3. Nom : `Soluble(s) MCP`  
4. URL : `https://mcp.csoluble.media` (une fois déployé)  
5. Connect  

Pose ensuite des questions naturelles :  
« Quelles solutions concrètes pour le zéro déchet ? »

**English**  
1. Open **Claude** (claude.ai or Desktop)  
2. Settings → Connectors → Add custom connector  
3. Name: `Soluble(s) MCP`  
4. URL: `https://mcp.csoluble.media` (once deployed)  
5. Connect  

Then ask natural questions:  
"What are the concrete solutions for zero waste?"

### Ce qu’il fait / What it does

**Français**  
- Recherche directe dans les 2 APIs du site (triptyque question/besoin/actions concrètes + contenu complet)  
- Affichage sous forme de belles cartes dans Claude  
- 12 prompts rapides (climat urgent, santé mentale, inclusion…)  
- Citations automatiques avec lien vers l’épisode  
- Mise à jour instantanée dès qu’un nouvel épisode sort  

**English**  
- Direct search in the site’s 2 APIs (question/need/actions triptych + full content)  
- Beautiful Episode Cards in Claude  
- 12 quick starter prompts (urgent climate, mental health, inclusion…)  
- Automatic citations with episode links  
- Instant update when a new episode is published  

### Tech / Technology

**Français**  
- MCP officiel (SDK Anthropic @modelcontextprotocol/sdk 1.27.1)  
- Déployé sur Netlify Functions  
- Connexion live aux APIs WordPress de csoluble.media  

**English**  
- Official MCP (Anthropic SDK @modelcontextprotocol/sdk 1.27.1)  
- Deployed on Netlify Functions  
- Live connection to csoluble.media WordPress APIs  

C’est un petit outil personnel que j’ai fait pour rendre le contenu de Soluble(s) plus accessible aux gens qui utilisent déjà l’IA au quotidien.  

This is a small personal tool I built to make Soluble(s) content more accessible to people who already use AI every day.
